# TDS v1.0

To compile, make sure you have MinGW installed as
well as MinGW development libraries (32-bit) for:<br>
* SDL2<br>
* SDL2_image<br>
* SDL2_mixer<br>
* SDL2_net<br>
* SDL2_ttf<br>

Then run:<br>
* make

Game.exe will be placed in the ./bin folder<br>
